# PhD Thesis Template

A LaTeX template for writing your PhD thesis at UTS.

## Features

- Pre-configured UTS formatting
- Chapter-based organization
- Bibliography management with BibTeX
- Optimized for VS Code + LaTeX Workshop

## Installation

### 1. Install TeX Live

Download and install [TeX Live](https://www.tug.org/texlive/) (full installation recommended).

### 2. Install VS Code Extensions

Install these extensions in VS Code:

- LaTeX Workshop
- LaTeX Utilities
- Code Spell Checker

### 3. Download the Template

[Download PhD Template :material-download:](https://github.com/o-niki-ri/QMGwiki/releases/download/v1.0/UTS-PhD-Template.zip){ .md-button .md-button--primary }
[View on GitHub :material-github:](https://github.com/o-niki-ri/QMGwiki/releases/latest){ .md-button }

## Usage

1. Open the template folder in VS Code
2. Edit `main.tex` and chapter files
3. Build with `Ctrl+Alt+B` (or `Cmd+Alt+B` on Mac)

## Backup Recommendations

!!! warning "Back up your work!"
    - Use Git for version control
    - Sync to cloud storage (OneDrive, Dropbox, etc.)
