# Getting Started

Welcome to the Quantum Materials Group! This guide will help you get set up.

## First Steps

1. **Request access** to the group's shared resources
2. **Set up your development environment** (VS Code, LaTeX, Python)
3. **Familiarize yourself** with group protocols and SOPs

## Essential Tools

| Tool | Purpose | Installation |
|------|---------|--------------|
| VS Code | Code editor | [Download](https://code.visualstudio.com/) |
| TeX Live | LaTeX distribution | [Download](https://www.tug.org/texlive/) |
| Python | Scripting | [Download](https://www.python.org/) |

## Recommended VS Code Extensions

- **LaTeX Workshop** — Build and preview LaTeX documents
- **LaTeX Utilities** — Additional LaTeX features
- **Code Spell Checker** — Catch typos in your documents
